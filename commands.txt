Собираем Docker-образ

	docker build -t qt-app

Создаем новый тег для существующего образа qt-app

	docker tag qt-app oleja4mo/alarmclock:1.0   

Запускает контейнер из образа, потребуется X-сервер
 
	docker run -it --rm -e DISPLAY=host.docker.internal:0 oleja4mo/alarmclock:1.0 python3 /tmp/main.py
